# Class System
- Added 'Institutional Knowledge' mechanic: each time a ruler learns a class, future rulers learning the same class will learn it quicker by starting with more initial progress points. Each completed study increases the starting points by 20, capping at 80.

# Interface
- Added in-game encyclopedia that covers mod-specific mechanics and allows the player to view such information in a more comprehensive fashion.
