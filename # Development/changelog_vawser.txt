#--------------------------------------
# Changelog
#--------------------------------------
# Countries
- Rastakhan of Zandalar is now considered Long-lived.
- Added portrait for Rastakhan.

# Disasters
- Rusty Empire:
 - Changed stability_cost_to_declare_war = 2 to stability_cost_modifier = 1.0
 - Added positive modifier upon finishing the disaster.

# Events
- Added time-gate to the Lei Shen resurrection event. Now only occurs after year 650.

# Balance
- Increased the Clique Influence boost to 3% per year (was 1%)
- Adjusted some sources of passive Clique Influence.

# Bugfixes
- Fixed Abyssal Tides spell applying Primordial Swell modifier.
- Fixed Living Flame: Common modifier using incorrect value for local_hostile_attrition.
- Fixed other countries that are not Azjol Nerub being able to use the Nerubian Tunnel Digging button.
- Fixed some Magic/Class/Zandalar variables being reset on game reload.
- Added clique safeguard to prevent a bug where clique could be lost from nations that use them.
- Fixed the Mission Tree view on the country selection screen.
